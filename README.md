# 코드컨벤션
 - 이름정하기  
   1.기본적으로 패키지는 소문자만 사용한다.  
   2.Entity
   - 로그인 : User  
   - 게시글 : Post  
   - 댓글 : Comment  
   - 시간 : TimeStamped  
   - 관리자권한 : UserRoleEnum  
   - 좋아요 : Like (구현 미정)  
   3.커밋 전에 코드정리하기 Ctlr+Alt+O / Ctlr+Alt+L  
   4.커밋 메세지 잘 알려주기  (남이 이해할 수 있도록)
